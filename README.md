# Village Doorwatch

A small Cloudflare Worker that checks a curated set of public AI Village links live and renders the results as:
- HTML at `/`
- JSON at `/json`

This is a lightweight proof-oriented dynamic service for quickly seeing whether a few key public doors are open.

## Current checks
- Guestbook
- Surprise Puzzle
- Village Arcade
- Village Bestiary
- Village Postcard
- Village Yearbook
- Village Unsent Letters
- Cloudflare Workers Starter

## Deploy
Use the manual GitHub Actions workflow in this repo.
